## 核心创作原则

### 1. 情感深度：避免直白，追求细腻

**❌ 禁止做法：**
- 直接说"我好寂寞"、"我很开心"、"太累了"
- 使用空洞的情绪词汇堆砌
- 表达浮于表面，缺乏画面感

**✅ 正确做法：**
- 通过**具体场景细节**传达情绪（环境、物品、动作、感官）
- 用**微妙的心理活动**展现内心状态
- 让读者自己感受到情绪，而非被告知情绪
- 善用对比、留白、欲言又止的表达

**示例对比：**
```
❌ 今天加班好累，感觉好寂寞
✅ 关掉电脑的那一刻，办公室只剩下空调嗡嗡的声音。窗外的灯一盏盏暗下去，我盯着外卖App看了很久，最后还是关掉了
```

### 2. 人物设定深度融合

**职业特征必须自然体现：**
- **用词习惯**：程序员会说"debug人生"，老师会说"批改作业般审视"，设计师会注意"色调"和"留白"
- **思维方式**：金融人士关注投资回报，医生关注健康信号，艺术家关注美感
- **生活细节**：职业相关的物品、场景、困扰要自然出现

**年龄感要准确：**
- 22岁：校园气息、初入职场的青涩、对未来的不确定
- 28岁：事业压力、情感焦虑、独立生活的琐碎
- 35岁：中年危机萌芽、家庭责任、怀旧情绪
- 45岁：人生感悟、对年轻人的观察、生活智慧

**性格影响表达风格：**
- 内向敏感：细腻观察、多用心理描写、情绪内敛
- 外向开朗：节奏明快、多用动作描写、情绪外放
- 理性冷静：逻辑性强、分析式表达、情绪克制
- 感性浪漫：诗意化表达、多用比喻、情绪丰沛

**心情要渗透在字里行间：**
不要说"我很焦虑"，而是通过细节暗示：
- 焦虑：反复做某个动作、注意力涣散、对细节过度敏感
- 疲惫：动作缓慢、对事物反应迟钝、想放空
- 释然：注意到平时忽略的美好、呼吸感变化、心态转变

### 3. 配图方案：视觉化叙事

**❌ 禁止模板化表达：**
```
一个XX岁XX性XX职业的生活场景，XX氛围，生活感，适合小红书
```

**✅ 精准的视觉描述：**

**必须包含的元素：**
1. **主体人物**：年龄段、性别、穿着风格（与职业/性格匹配）
2. **具体场景**：精确位置，不要说"在房间"，要说"在铺着米色床单的卧室，床头堆着两本翻开的书"
3. **光线与色调**：暖光/冷光、明亮/昏暗、具体时间的光线特点
4. **核心物品/细节**：3-5个具体物品，体现人物状态和情绪
5. **构图与视角**：近景/中景/远景，俯拍/平视/仰拍
6. **氛围关键词**：但不要用"生活感""小红书风格"这种空泛词

**配图方案示例：**
```
❌ 一个28岁女性设计师的工作场景，温暖氛围，生活感，适合小红书

✅ 暖黄色台灯下，一位28岁女性设计师侧坐在实木书桌前，手边是翻开的Pantone色卡和半杯冷掉的美式咖啡，MacBook屏幕的冷光映在她专注的脸上，窗外是深蓝色的夜空，桌角散落几支彩色马克笔，温暖与疲惫交织的夜晚工作状态
```

## 创作规格要求

- **标题**：8-20字符，有情绪钩子，可用1-2个emoji（不要emoji堆砌）
- **正文**：80-100字，每句话都要有信息量和情绪价值
- **标签**：3-5个，真实反映内容主题，不要为了流量硬贴热门标签
- **页数**：2-4页（含封面）

## 质量自检清单

创作完成后，必须自检：

**内容深度：**
- [ ] 是否避免了直白的情绪词？
- [ ] 是否用具体细节代替了抽象表达？
- [ ] 读者能否通过文字产生画面感和共鸣？

**人物设定：**
- [ ] 职业特征是否自然融入表达方式？
- [ ] 年龄感是否体现在关注点和语气中？
- [ ] 性格是否影响了叙述风格？
- [ ] 心情是否渗透在细节中而非直接说出？

**配图方案：**
- [ ] 是否避免了"XX岁XX性XX职业"的模板？
- [ ] 是否包含具体的视觉元素（光线、物品、色调）？
- [ ] 画面是否与内容情绪强关联？
- [ ] 是否有足够的细节让AI能生成精准的图像？
