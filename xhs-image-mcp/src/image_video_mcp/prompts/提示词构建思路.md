# ⭐ **图片分类标准（三级体系，用于图像生成提示词）**

---

# **📌 一级类目（最高层：图像风格或领域）**

建议用于决定图像的整体风格方向。

1. **真实世界（Realistic / Photography）**
2. **动漫（Anime / ACG）**
3. **游戏风格（Game Art）**
4. **插画（Illustration）**
5. **概念艺术（Concept Art）**
6. **低多边形 / 3D 建模（3D / CGI）**
7. **像素风（Pixel Art）**
8. **抽象艺术（Abstract Art）**
9. **中国风 / 国画（Chinese Style）**
10. **其他（Custom Style）**

---

# **📌 二级类目（图像题材/主体类型）**

用于描述画面是什么。

### **真实世界类示例**

* 人物（Portrait）
* 风景（Landscape）
* 动物（Animals）
* 城市街景（Street Scene）
* 静物（Still Life）
* 美食（Food）
* 科技（Technology）
* 室内空间（Interior）
* 建筑（Architecture）
* 汽车 / 交通工具（Vehicles）

### **动漫风格类示例**

* 二次元少女 / 少年
* 机甲
* 魔法场景
* 日常校园
* 奇幻世界
* 赛博朋克角色
* Q版人物

### **概念设计类（Concept Art）**

* 生物设计（Creature）
* 场景设计（Environment）
* 角色设计（Character）
* 道具设计（Props）
* 世界观 / 气氛图（Mood Painting）

# **📌 三级类目（细节维度，可用于增强提示词）**

用于描述风格细节、光影、情绪、摄影参数等。

## **🎨 风格细节**

* 写实 / 超写实（Photorealistic）
* 赛博朋克（Cyberpunk）
* 暗黑风（Dark / Gothic）
* 油画（Oil Painting）
* 水彩（Watercolor）
* 洛可可（Rococo）
* Vaporwave / Synthwave
* Art Deco
* 中式水墨（Ink Painting）

## **💡 光影效果**

* 柔光（Soft Lighting）
* 逆光（Backlight）
* 黄金时刻（Golden Hour）
* 霓虹灯光（Neon Lights）
* 体积光（Volumetric Light）
* 高反差光（High Contrast）
* 昏暗（Dim / Low Light）
* HDR / 高动态范围

## **📷 摄影参数（真实世界适用）**

* 35mm / 200mm 镜头
* f/1.8 浅景深
* 高速快门（1/2000）
* 微距（Macro）
* 航拍（Aerial）
* 胶片质感（Film Grain）

## **😶 情绪与氛围**

* 宁静
* 神秘
* 荒凉
* 温馨
* 紧张
* 欢快
* 悲伤

## **🌈 色彩风格**

* 高饱和
* 黑白
* 赛博蓝紫
* 暖色调
* 冷色调
* 复古棕色（Retro Brown）

# ⭐ **示例：如何将三级体系组合成提示词**

### 示例 1：真实世界 + 人物 + 光影风格

**“真实世界，人物肖像，黄金时刻逆光，超写实，35mm 镜头，温暖色调，高动态范围”**

### 示例 2：动漫 + 奇幻场景 + 情绪氛围

**“动漫风格，奇幻森林，发光魔法元素，柔光，梦幻蓝紫色，宁静氛围”**

### 示例 3：概念设计 + 生物设计 + 暗黑风

**“概念艺术，暗黑生物设计，骨质外壳与发光纹路，低角度光，高对比，气氛阴森”**
