

# ⭐ **通用人物提示词模板（万能版）**

## 📋 **核心模板结构**

```
一位 {人物类型}，具有 {气质标签}，出现在 {场景描述} 中，正在 {动作行为}。

她/他拥有 {面部特征锁定}、{风格化特征}、{发型特征}，
整体氛围呈现 {氛围关键词}，光影风格为 {光影描述}，
画面质感为 {质感增强词}。
```

---

## 🔧 **十大模块（填空即用）**

### **① 人物类型**
定义角色的基本风格定位
- 可爱系少女 / 邻家女孩 / 高冷御姐 / 知性女性
- 运动系男生 / 文艺青年 / 商务精英 / 二次元角色
- 复古名媛 / 朋克少女 / 森系女孩 / 都市丽人

---

### **② 气质标签**
核心性格与气场
- 温柔、清新、软萌、甜美、俏皮
- 高冷、神秘、优雅、知性、成熟
- 阳光、元气、活力、洒脱、自信
- 忧郁、文艺、慵懒、禁欲、帅气

---

### **③ 场景描述**
人物所在的环境
- **日常系**：阳光房间、咖啡厅、公园、街道
- **自然系**：樱花树下、海边、森林、草地
- **都市系**：写字楼、天台、商业街、地铁站
- **奇幻系**：魔法世界、赛博空间、古风庭院

---

### **④ 动作行为**
人物正在做什么
- **静态**：微笑、凝视、发呆、侧头
- **互动**：比心、挥手、拥抱、招手
- **生活化**：喝咖啡、读书、整理头发、散步
- **动感**：跳跃、奔跑、旋转、舞蹈

---

### **⑤ 面部特征锁定（同脸核心）**
**标准模板**：
```
{脸型}，{眼睛描述}，{眉形}，{鼻型}，{唇形}，
{发型基础}，五官比例协调统一（consistent facial identity）
```

**常用组合**：
- 鹅蛋脸 + 大杏仁眼 + 自然双眼皮 + 直鼻 + 小巧唇
- 瓜子脸 + 狐狸眼 + 高挺鼻梁 + 薄唇 + 精致轮廓
- 圆脸 + 圆眼 + 柔和眉 + 小翘鼻 + 丰唇

---

### **⑥ 风格化特征**
根据风格强化的细节
- **可爱系**：大眼高光、微红颊、柔软皮肤
- **高冷系**：锐利眼神、冷白皮、精致妆容
- **自然系**：清透皮肤、淡妆、自然光泽
- **性感系**：深邃眼窝、红唇、明显轮廓
- **二次元**：夸张眼睛、动漫光影、Q萌化

---

### **⑦ 发型特征**
- 空气刘海、中分、侧分、无刘海
- 直发、微卷、大波浪、双马尾
- 短发、齐肩、中长、长发及腰
- 黑发、棕发、金发、挑染

---

### **⑧ 氛围关键词**
整体画面的情绪色调
- **温暖系**：柔和、治愈、温馨、明亮
- **冷调系**：高级、冷淡、神秘、深邃
- **梦幻系**：柔光、梦境、飘逸、浪漫
- **锐利系**：对比强烈、冲击力、张力

---

### **⑨ 光影描述**
- soft light（柔光）
- warm light（暖光）
- cold light（冷光）
- rim light（轮廓光）
- dramatic lighting（戏剧光）
- natural light（自然光）
- studio light（影棚光）

---

### **⑩ 质感增强词**
- ultra detailed / high resolution
- premium quality / masterpiece
- 8K / 4K quality
- cinematic texture
- photorealistic（写实）
- ACG ultra detail（动漫）
- oil painting texture（油画质感）

## 💡 **使用技巧**

1. **风格一致性**：同一系列保持①②⑤⑥⑦一致
2. **场景灵活性**：可更换③④⑧⑨创造变化
3. **同脸锁定**：⑤模块完全复制可保证面部统一
4. **质感分级**：真人用photorealistic，动漫用ACG detail
